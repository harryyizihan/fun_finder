# funFinder

www.iamsobored.net

## Description
Activity Recommendation Web Application

This web app helps people escape the boring life based on their locations nearby. You can even store the activity for later by simply dropping the favorite button!

Those favorited activities would be permanently stored in the back-end database for your future visit!

What's more, recommendation would boost your search based on your favorited events! NO need to SIGN UP, TRY it now!"
